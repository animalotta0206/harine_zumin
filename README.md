## Pythonの導入
このbotはPythonで稼働しております。<br>
このコードをそのまま利用する場合には、Pythonの環境が必要ですので各自ご準備ください。<br>
Pythonのインストールは<br>
```sh
$ sudo apt-get install python3
```
で可能です。<br>
インストールされたかどうかを確認するには、<br>
```sh
$ python -- version
```
を実行してください。<br>
